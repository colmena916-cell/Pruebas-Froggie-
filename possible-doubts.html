Under the Hood: Connection & Intelligence (API Keys) 🔑🤖

Currently, Froggie AI does not host its own large language model due to the high costs of server infrastructure. Instead, the platform acts as a bridge, utilizing API Keys that you provide to connect with some of the best AI models on the market. While our ultimate long-term dream is to host a dedicated, custom-tuned model optimized for roleplay, here is a quick summary of how to connect each provider from your settings panel:

✓ Recommended

1. OpenRouter (Advanced Roleplay Models)

This is the absolute star of the house if you are looking for an immersive narrative experience. It allows you to invoke amazing models like Mythomax or Anthropic's Claude variants. To set it up:

  • Create an account at openrouter.ai.
  • Generate a new API Key in your dashboard and paste it into the designated field inside Froggie AI.
  • Don't forget to type the exact model identifier you want to call (e.g., gryphe/mythomax-l2-13b) in your settings panel so it routes properly!
✓ Available

2. OpenAI (GPT-3.5 / GPT-4)

Perfect for fast, highly structured responses with great logic. It requires you to generate a token (starting with sk-...) from your OpenAI developer console and ensure your billing account has an active balance to prevent quota limit errors.

⚠ Currently Unavailable

3. Google Gemini (Under Maintenance)

While you will see the visual fields ready to receive Google Gemini keys in the user interface, we want to clarify that this model is currently not active within the chat rooms. We are restructuring our backend code to integrate it properly without disrupting the prompt parameters. We'll let you know as soon as it's ready to jump into action!

🔧 Flexible

4. Custom API (Advanced Users)

This option lets you connect any compatible AI provider of your choice by entering a custom base URL and its corresponding API key. It is designed for users who already work with alternative or self-hosted models not listed above.

If you are unsure about the exact URL your provider uses, don't worry! You can simply ask any AI assistant (such as ChatGPT, Claude, or Gemini) something like: "What is the base API URL for [provider name]?" and it will point you in the right direction in seconds. 🤖💡

Step-by-Step Guide: Bringing a Character to Life 🎨✍️

Designing a bot on Froggie AI is a magical process, but it requires structure to help the AI accurately capture the exact persona you have in mind. Here is a breakdown of our official creation form:

Step 1: Basic Identity & Visuals

The very first thing other users will see. Here, you need to define your character's avatar or photo (we highly recommend using a direct, stable image URL), assign their official name, and write a brief subtitle that summarizes who they are in a single sentence. These initial fields have strict character limits to keep the card layouts clean.

Reference Image: primera parte del bot.png

Step 2: Public Description

This field serves as a compact summary of who your character is. Think of it as their identity at a glance: include key aspects of their physical appearance and their most defining personality traits. The key here is to be clear, direct, and concise — a few well-chosen details paint a much stronger picture than a long vague paragraph. This section is also visible to the public, so it doubles as the first impression readers get of your creation!

Reference Image: segunda parte.png

Step 3: Deep System Definition (The Soul of the Bot)

This is the most critical and hidden part of the setup form. Here, your writing needs to be extremely clear, direct, and detailed. Use explicit attributes and lore traits to guide the AI on how to act, its backstory, relationship to the user, speech mannerisms, and psychological traits.

Structuring this section with clear trait lists or system blocks helps the underlying language model absorb the character personality without getting confused during long chats.

Sequential Reference Images: tercera parte.png, cuarta parte.png, and quinta parte.png

Step 4: First Message & Visibility

To wrap things up, you must write the bot's First Message / Greeting. This is vital because it not only triggers the roleplay session, but it also subtly dictates the writing style to the AI (for example, if it should use asterisks *actions* for physical movements or quotation marks for dialogue speech).

Lastly, select the privacy setting: Public (so it appears on the global community feed) or Private (exclusively accessible for you to chat from your profile layout).

Reference Image: ultima parte.png

💡 Quick Pro-Tip: While this structure is highly recommended to prevent bots from losing their memory tracking, Froggie AI is all about creative freedom! Anyone is welcome to experiment with different definition styles and prompt formats to discover unique interactions.
